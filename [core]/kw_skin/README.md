# kw_skin

## Requirements
- [skinchanger](https://github.com/KW-Org/skinchanger)

## Download & Installation

### Using [fvm](https://github.com/qlaffont/fvm-installer)
```
fvm install --save --folder=esx esx-org/kw_skin
```

### Using Git
```
cd resources
git clone https://github.com/KW-Org/kw_skin [esx]/kw_skin
```

### Manually
- Download https://github.com/KW-Org/kw_skin/archive/master.zip
- Put it in the `[esx]` directory

## Installation
- Import `kw_skin.sql` to your database
- Add this in your `server.cfg`:

```
start kw_skin
```
